## Detailed Step-by-Step Flow

### 1. User Registration & Profile Setup
*   **Sign Up:** The customer opens the application and registers by creating an account.
*   **Body Metrics Input:** During profile initialization, the user provides personal parameters including age, weight, height, gender, and activity level.
*   **Profile Management:** Once registered, these metrics can be viewed and updated at any time from the Profile dashboard.

### 2. Diet Input & Analysis
*   **Logging:** On the main dashboard, the user chooses to input their current dietary details or request suggestions.
*   **Analysis:** The application calculates Basal Metabolic Rate (BMR) and Total Daily Energy Expenditure (TDEE) based on the user's recorded parameters.

### 3. Nutrition Planning & Execution
*   **Suggestions:** The system generates daily calorie counts and macro targets (protein, carbs, fats) alongside tailored meal ideas.
*   **Implementation:** The user implements the recommended meal structure and tracking protocols in their daily life.
*   **Follow-Up & Monitoring:** The system tracks progress against historical suggestions, allowing the user to view generated plans on the dashboard.

### 4. Evaluation Loop
*   **Metric Review:** The user checks whether weight loss, gain, or maintenance targets are met.
*   **Adjustment:** If goals are not yet reached, the flow loops back to generating new parameters and nutrition suggestions based on updated weight and metrics.
*   **Success:** When targets are successfully achieved, the cycle completes.
